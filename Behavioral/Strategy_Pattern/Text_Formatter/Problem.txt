You are building a text formatting tool. 
The system should allow different text formatting strategies that can be swapped at runtime.

Requirements:

The system should support at least three formatting strategies:

1. UpperCaseFormatter → converts text to uppercase.

2. LowerCaseFormatter → converts text to lowercase.

3. CapitalizeFormatter → capitalizes the first letter of each word.

A TextEditor class should use a Formatter strategy to format text.

The editor should be able to switch between strategies dynamically.

The client code should demonstrate switching between different strategies without modifying the TextEditor class itself.

- TextEditor
    - strategy

    + set_strategy(strategy: Strategy)->Stragy
    + format_it(txt)->str => strategy.formatting(txt)

- Format
    + formatting(txt: str)->str

- UpperCaseFormatter(Format)

- LowerCaseFormatter(Format)

- CapitalizeFormatter(Format)
